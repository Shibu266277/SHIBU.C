Write a code to get the input in the given format and print the output in the given format.

Input Description:
A single line contains a string.

Output Description:
Print the characters in a string separated by line.

Sample Input :
wellcomeyouall
Sample Output :
w
e
l
l
y
0
u
a
l
l

code:
H = input().strip()
for A in H:
    print(A)
